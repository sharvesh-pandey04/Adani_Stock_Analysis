# Adani Stock Analysis Project 📈

This project analyzes the stock market data of Adani Enterprises using Python libraries like **Pandas**, **NumPy**, and **Matplotlib**.

## Key Features:
* Data cleaning (Handling NaN values and header fixes).
* Mathematical analysis using NumPy (Mean, Max, Min prices).
* Monthly average calculations using GroupBy.
* Exporting cleaned data to CSV.

## Technologies Used:
* Python
* Jupyter Notebook
* Pandas
* NumPy
